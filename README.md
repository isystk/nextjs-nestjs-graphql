## 📗 プロジェクトの概要

Next.js（フロントエンド）・Nest.js（サーバーサイド）・GraghQLを利用したサンプルアプリケーションです。


### 利用している技術

- Next.js 10 (React.js 16)
- typescript 4
- material-ui
- redux-toolkit
- aws-cognito
- aws-lambda
